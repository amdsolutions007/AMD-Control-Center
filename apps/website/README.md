# AMD AI NEXUS (Website)

Next.js App Router site for AMD Media Solutions.

## Run

1. Install Node.js (LTS) and npm.
2. From this folder:

- `npm install`
- `npm run dev`

## Notes

- Add your hero video as `public/hero.mp4` (optional).
- Footer/contact info is auto-populated from `../../config/social_links.json` and `../../config/billing_profile.json`.
